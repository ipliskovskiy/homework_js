Первые три команды alert выведут код функции:
function a() {
 alert(this);
}
Вывод произойдет, потому-что мы не указали "()" тоесть не вызываем функцию. Так же при вызове или выводе "a" - мы не выведем/вызовем и эту функцию:
function a(x) {
 return x * 2;
}
Первая функция в коде, указана a(x), вторая функция в коде a(). JS последовательно загружает все функции сверзу вниз. Так как в JS нет перегрузки, JS переопредели первую загруженныую функцию a(x) как на a(); Так же у нас есть var объявляющий переменную a, но не инициализирующую ее. Значение в переменной a не меняется.

b - обявлена выше как глобальная переменная и присвоенна ей функция a()
b(1,2,3) - при вызове отправит в функцию:
