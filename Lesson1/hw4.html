Задача:

Самостоятельно разобраться с атрибутами тега script (async и defer)

Ответ:

defer - используется в тэге script. Без параметра src игнорируется. Когда используется defer, наши скрипты загружаются поочередно.

async - скрипт загружается в любое время без каких либо условий. Скрипты могут запускаться параллельно.
