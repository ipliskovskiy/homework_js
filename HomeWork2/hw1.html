На главную

Дан код:

var a = 1, b = 1, c, d;

c = ++a; alert(c); // 2

d = b++; alert(d); // 1

c = (2+ ++a); alert(c); // 5

d = (2+ b++); alert(d); // 4

alert(a); // 3

alert(b); // 3

Почему код даёт именно такие результаты?

Ответ:

var a = 1, b = 1, c, d;

c = ++a; alert(c); // 2

Прединкремент - в начале a увеличиваем на единицу и потом записываем значение в "c".

d = b++; alert(d); // 1

Постинкремент в начале «b» записываем значение в «d», а после b увеличиваем на единицу.

c = (2+ ++a); alert(c); // 5

Изначально «a» = 2. Перовым делом из-за прединкремента «a» увеличивается на единицу (3), после выполняем суммирование (2 + 3). Далее сумму (5) записываем в «c».

d = (2+ b++); alert(d); // 4

Изначально «b» = 2. Перовым делом выполняем суммирование (2 + b) = 4, далее выполняем постинкремент «b». «b» увеличивается на единицу (3). Далее сумму записываем в «d». Так как мы складывали 2 и b до инкремента, мы сложили 2 + 2. «d = 4»

alert(a); // 3

Изначально a = 1

Был один прединкремент a = 2

Далее один постинкремент a = 3

alert(b); // 3

Изначально b = 1

Был один постинкремент b = 2

Далее один прединкремент b = 3
